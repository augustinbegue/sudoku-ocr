# Sudoku OCR

## Usage

```
./main [options] file ...
Options:
        -o <file>     Save the output into <file>.
        -m            Save the mask image into <file>.
```

## Build

- Make sure you've installed ``make``, ``gcc``, ``sdl2`` and ``sdl2_image``

- Clone the repo

- Execute ``make`` at the root of the repo
